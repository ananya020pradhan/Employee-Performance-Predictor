# 🚀 Employee Performance Predictor using Data Analytics

## 📌 Project Overview

The **Employee Performance Predictor** is an AI-powered system that analyzes employee-related data and predicts whether an employee is likely to be a **high performer or low performer**.

This project demonstrates how **data analytics and machine learning** can support **HR decision-making**, improve workforce productivity, and optimize organizational performance.

---

## 🎯 Problem Statement

Organizations often struggle to:

* Identify high-performing employees early
* Detect underperformers
* Make data-driven promotion decisions
* Plan training and development strategies

This project solves these problems using a **machine learning-based predictive model**.

---

## 💡 Business Value

* 📈 Improves employee performance management
* 🧠 Enables data-driven HR decisions
* 🎯 Identifies high-potential employees
* 📉 Helps reduce employee attrition
* 📊 Supports training & development planning

---

## 🧠 Machine Learning Approach

* Problem Type: **Binary Classification**
* Models Used: **Random Forest Classifier**
* Data Type: **Structured (Tabular Data)**

### Features Used:

* Age
* Experience
* Salary
* Training Hours
* Projects Completed
* Attendance Rate
* Department

### Target:

* Performance (0 = Low, 1 = High)

## ▶️ How to Run the Project

### 1. Clone the repository

```
git clone https://github.com/YOUR_USERNAME/Employee-Performance-Predictor.git
cd Employee-Performance-Predictor
```

### 2. Create virtual environment

```
python3 -m venv venv
source venv/bin/activate
```

### 3. Install dependencies

```
pip install -r requirements.txt
```

### 4. Run the ML pipeline

```
python main.py
```

### 5. Run the Streamlit App

```
streamlit run app.py
```
